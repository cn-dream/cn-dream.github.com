布局和样式基础

flex-direction

决定元素的排列方式

flex-wrap

决定元素如何换行（排列不下时）

flex-flow

是 flex-direction 和 flex-wrap 的简写 。
例如: flex-flow row wrap

justify-content

元素在主轴上的对其方式

align-items

元素在交叉轴的对齐方式

flex-grow

当有多余空间时，元素的放大比例

flex-shrink

空间不足时，元素的缩小比例

flex-basis

元素在主轴上占据的空间

flex

是 flex-grow , flex-shrink, flex-basis 的简写

order

定义元素的排列顺序

align-self

定义元素自身的对齐方式
